# Raccourcis clavier

::: info
Sur macOS, `⌘` ; sur Windows/Linux, `Ctrl`.
:::

## Navigation

| Raccourci | Action |
|---|---|
| `⌘K` | Ouvrir la palette de commandes |
| `⌘,` | Ouvrir les réglages |

## Projet

| Raccourci | Action |
|---|---|
| `⌘N` | Nouveau projet |
| `⌘O` | Ouvrir un projet |
| `⌘⇧W` | Fermer le projet |

## Agent

| Raccourci | Action |
|---|---|
| `⌘Enter` | Envoyer le message |
| `Tab` | Basculer Plan / Build |
