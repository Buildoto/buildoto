# Introduction

Buildoto est une application de bureau qui combine un **agent IA**, **FreeCAD** comme moteur de modélisation 3D, et **Git + GitHub** pour versionner vos projets de construction.

Elle s'adresse aux professionnels du BTP, bureaux d'étude, architectes et ingénieurs qui veulent une alternative **locale**, **open-source** et **versionnable** aux suites BIM classiques.

## Pourquoi Buildoto ?

- **Pilotage par agent** : décrivez en langage naturel, l'agent exécute les outils FreeCAD structurés (23 outils couvrant Arch, Part, Draft, Spreadsheet).
- **Multi-providers** : Claude, GPT, Mistral, Gemini, Ollama (local), ou Buildoto AI.
- **Git first** : chaque modification peut devenir un commit, push sur GitHub en un clic.
- **Local par défaut** : tout tourne sur votre machine, aucun backend propriétaire requis.
- **Extensible** : serveurs MCP pour connecter l'agent à vos outils métiers.
