# 01 — Cube

**Niveau** : débutant
**Outils** : `part_create_box`
**Durée** : < 1 minute

## Objectif

Vérifier en 30 secondes que :
1. Le sidecar FreeCAD démarre
2. L'agent peut appeler un outil structuré
3. La géométrie s'affiche dans le modeleur

## Prérequis

- Clé Anthropic configurée (ou tout autre provider)
- Un projet Buildoto ouvert (vide ou non)

## Résultat attendu

Un cube de 2 m × 2 m × 2 m dans le modeleur, et un fichier `.FCStd` créé dans le projet.
